# What is Git?

Git is a distributed version control system used to track changes in code.
It lets you:

- Maintain history (commits)
- Work in parallel (branches)
- Collaborate without overwriting each other

---

How to Push Code to GitHub

# 1. Initialize repo
git init

# 2. Add files
git add .

# 3. Commit changes
git commit -m "initial commit"

# 4. Link GitHub repo
git remote add origin https://github.com/username/repo.git

# 5. Push code
git branch -M main
git push -u origin main
